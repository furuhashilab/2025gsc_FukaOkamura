# 2025年ゼミ論/中間発表


## テーマ
今後の地理データ形式の在り方について：Shapefile と GeoJSON の比較
## 概要
本研究では、GIS分野で長年利用されてきた Shapefile と、Web技術の普及によって急速に利用が広がった GeoJSON の2つの地理データ形式を比較し、Shapefile派と廃止派の意見を聞き、今後のファイル形式の在り方について考察する。Shapefile は複数の拡張子で構成されており、文字コードに Shift-JIS が含まれる場合は、読み込み時に文字化けが発生するなど、操作が煩雑であるという問題がある。一方、GeoJSON は1ファイルで完結し、UTF-8 を前提としているため文字化けの心配が少なく、QGISやGitHubとの互換性が高い。筆者自身の初学者としての経験においても、Shapefile の複雑さが学習の障壁となった一方、GeoJSON はスムーズに扱える形式であった。本研究は、この体験を起点に、文献調査・ヒアリングを通じてファイル形式の在り方を考察すると共に、行政オープンデータ提供の改善に資する知見を提供することを目的とする。
