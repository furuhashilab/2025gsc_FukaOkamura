# 2025ゼミ論_地理空間データ形式に着目したCKANベース・オープンデータポータルの国際比較

## 概要

本研究は、オープンソースのデータポータル基盤である CKAN において提供されている地理空間データ形式に着目し、日本の都道府県および海外の国単位で、その提供実態を比較・分析するものである。

分析にあたっては、CKAN 上で提供されているすべてのファイル形式を網羅的に扱うのではなく、位置情報を含む代表的な地理空間データ形式（GeoJSON、SHP、KML、GeoPackage）と、位置情報を含まない主要な非空間データ形式（Excel、PDF、DOCX、CSV）の計8種類に対象を限定した。これにより、地理空間データがオープンデータ全体の中でどのように位置づけられているのかを、相対的に把握することを試みた。

本研究の目的は、CKAN における地理空間データ形式の提供状況が、国・自治体ごとのオープンデータ政策の成熟度や、データ利活用に対する設計思想をどのように反映しているのかを明らかにすることである。

---

## I. Introduction（研究背景・目的）

近年、オープンデータは行政の透明性向上を目的とした情報公開の手段にとどまらず、民間活用やデータ分析を通じた社会課題解決の基盤として、その重要性を高めている。特に、地理空間データは都市計画、防災、環境政策、交通分野など、多様な行政分野において活用が期待される基盤的情報である。

こうしたオープンデータの公開と利活用を支える代表的な基盤の一つが、世界的に利用されているオープンソースのデータポータルソフトウェアである CKAN である。CKAN は多くの国や自治体で導入されている一方で、その上で提供されているデータの内容や形式には、国や自治体ごとに大きな差が存在する。

特に、地理空間データの提供形式に着目すると、日本の自治体では CSV や PDF といった人間による閲覧を主眼とした形式が多く見られる一方で、GIS での分析やシステム連携を前提とした GeoJSON や SHP といった形式の提供は限定的であるとの指摘がなされている。しかし、こうした傾向を定量的に示し、海外事例と比較した研究は十分とは言えない。

そこで本研究では、CKAN 上で提供されている地理空間データ形式の提供数に着目し、日本の都道府県および海外諸国を対象とした比較分析を行う。すべてのファイル形式を網羅的に扱うのではなく、位置情報を含むデータ形式と含まないデータ形式を限定的に設定することで、オープンデータにおける地理空間データの相対的な位置づけを明らかにする。

本研究が設定する問いは以下の通りである。

> **CKAN における地理空間データ形式の提供数は、  
> 国・自治体のオープンデータ政策の成熟度を反映しているのか。**

本研究を通じて、オープンデータを「公開すること」を主眼とする政策と、「再利用・分析」を前提とした政策の違いを、地理空間データ形式という技術的側面から読み解くことを目的とする。

---

## II. Methods（手法）

本研究では、CKAN を基盤とするオープンデータポータルを対象とし、以下の手法で分析を行った。

### 1. 調査対象
- **日本**：都道府県単位の CKAN ベースのオープンデータポータル  
- **海外**：国単位の CKAN ベースのオープンデータポータル  

### 2. 分析対象データ形式
- 地理空間情報を含むファイル形式  
  - GeoJSON（JSON）
  - SHP  
  - KML  
  - GeoPackage  

### 3. 分析方法
- 各ポータルにおいて、位置情報を含むデータセット数を形式別に集計
- 日本は都道府県別、海外は国別に集計結果を整理
- 集計結果をスプレッドシートにまとめ、提供数および形式構成比を比較分析した

---

## III. Results（結果）

分析の結果、以下の点が明らかになった。

- 日本では、都道府県によって GeoJSON や SHP といった地理空間データ形式を全く提供していない自治体が存在した。
- 多くの自治体において、CSV 形式のデータ提供数は多い一方で、GIS での利用を前提とした GeoJSON や SHP の提供数は限定的であった。
- 都道府県間で、提供されているデータセットの総数に大きな差が見られた。

一方、海外事例、特にオーストラリア、アメリカでは以下の特徴が確認された。

- CKAN 上で提供されているデータ総数が多い
- GeoJSON、SHP、GeoPackage、ArcGIS REST API など、複数の地理空間データ形式が豊富に提供されている
- 地理空間データが一定の割合を占めている

---

## IV. Discussion（考察）

これらの結果から、日本と海外のオープンデータ政策には、設計思想の違いが存在する可能性が示唆される。

日本の自治体においては、オープンデータが「公開される成果物」として位置づけられており、説明責任や情報公開を主目的としたデータ提供が中心となっていると考えられる。その結果、CSV 形式のように人間が閲覧しやすいデータは多いものの、分析や再利用を前提とした地理空間データ形式の提供は十分ではない傾向が見られた。

一方、オーストラリアをはじめとする海外の事例では、地理空間データが行政運営の基盤情報として位置づけられ、API 連携や分析利用を前提とした形式で提供されている。このことは、オープンデータを「社会インフラ」として捉える姿勢の表れであると考えられる。

---

## V. Conclusion（結論）

本研究の分析から、CKAN における地理空間データ形式の提供状況は、国・自治体ごとのオープンデータ政策の成熟度や、データ利活用に対する考え方を一定程度反映している可能性が示された。

日本のオープンデータ政策は、データの「公開」に重点が置かれている一方で、「再利用」や「分析」への最適化は依然として十分ではない。一方、海外では地理空間データを前提とした実務志向のデータ提供が進んでいる。

今後は、オープンデータを単に公開するだけでなく、どのような利用者を想定し、どのような形式で提供すべきかという視点を政策設計に取り入れることが重要である。

## VI. Limitations and Future Work（研究の限界と今後の課題）

本研究は、CKAN 上で提供されている地理空間データ形式の提供実態を定量的に把握することを目的としたが、分析対象および手法にはいくつかの制約が存在する。

第一に、本研究では CKAN 上で提供されているすべてのファイル形式を網羅的に集計したわけではない。分析対象は、位置情報を含む代表的な地理空間データ形式（GeoJSON、SHP、KML、GeoPackage）と、位置情報を含まない主要な非空間データ形式（Excel、PDF、DOCX、CSV）の計8種類に限定している。そのため、その他の形式（例：画像形式や独自フォーマット等）は分析に含まれておらず、CKAN 全体におけるファイル形式の分布を完全に反映しているとは言えない。

第二に、本研究で算出した割合は、「位置情報を含むデータ形式4種」と「位置情報を含まないデータ形式4種」という限定的な分類に基づくものであり、すべてのデータセットを母集団とした厳密な構成比ではない。この点は、結果の解釈において留意すべき重要な前提条件である。

第三に、本研究ではデータセットの提供数に着目して分析を行っており、各データの内容や品質、更新頻度、空間精度、実際の利用状況といった質的側面については十分に検討できていない。地理空間データの実用性をより正確に評価するためには、これらの要素を含めた分析が必要である。

今後の研究では、分析対象とするファイル形式を拡張し、CKAN 上で提供されている全形式を網羅的に整理した上で、地理空間データの位置づけをより精緻に評価することが求められる。また、ファイル形式の選択が、各国・自治体のオープンデータ政策や想定利用者像とどのように結びついているのかを、政策文書の分析や事例調査を通じて検証することが今後の課題である。
